# AI-Based Personalized Diet Recommendation System

This project is a machine learning-based diet recommendation system that predicts daily calorie requirements and generates a personalized diet plan.

## Technologies Used
- Python
- Tkinter (GUI)
- Pandas
- Scikit-learn (Linear Regression)

## Features
- Predicts calorie needs using ML
- Calculates BMI
- Suggests personalized diet plan
- Desktop GUI application

## How to Run

1. Install dependencies:
   pip install pandas scikit-learn

2. Train model:
   python train_model.py

3. Run application:
   python Dietplan.py
